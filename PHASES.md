## Фаза 6: Доработки по режимам редактирования и автосохранению

- [x] Исправить режимы редактирования заметок (убрать split режим, оставить только просмотр/редактирование)
- [x] Исправить отображение текста в режиме редактирования (добавлена панель форматирования, исправлено обновление контента)
- [x] Улучшить логику автосохранения версий (`NoteVersionService`):
  - если с момента последнего `updatedAt` заметки прошло меньше N минут — **не создавать** новую версию (версии неизменяемы, обновляется только заметка);
  - если содержимое (title/content) совпадает с предыдущей версией — **не создавать** версию (например, пробел и затем backspace);
  - после истечения N минут с последнего `updatedAt` заметки — создавать новую версию со снимком состояния **до** текущего сохранения
- [x] Вынести в параметры интервалы автосохранения и версионирования:
  - **бэкенд:** поля `autosave_delay_seconds`, `version_consolidation_window_minutes` в `users`; системные defaults — env (`AUTOSAVE_DELAY_SECONDS`, `NOTE_VERSION_CONSOLIDATION_WINDOW_MINUTES`); `UserSettingsResolver` + `PATCH /api/auth/settings`;
  - **фронтенд:** `SettingsView` (`/settings`), селекты (автосохранение: 5–60 сек; версионирование: 1–60 мин); пустое значение → NULL (системный default); переход по клику на email в navbar
