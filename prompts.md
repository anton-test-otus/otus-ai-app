## Промпт 2

> Персональная база знаний с markdown и живым предпросмотром. Заметки с иерархией, версионированием (историей), wiki-ссылками и тегами. Корзина для хранения заметок 30 дней после удаления. Drag-and-drop заметок между папками. Автосохранение с уведомлением.
>
> Технологии: Vue + PHP фреймворк (на данный момент предполагаю Symfony, но можно предложить что-то лучше для этого типа приложения).
>
> База данных: PostgreSQL
>
> Аутентификация: многопользовательская

### Выполненные действия:
- Сформирован план архитектуры приложения
- Определён стек технологий:
  - Backend: Symfony 7, API Platform, Doctrine, JWT
  - Frontend: Vue 3, Vite, Pinia, Milkdown
  - База данных: PostgreSQL 16
- Спроектирована модель данных (User, Note, Folder, Tag, NoteVersion, NoteLink)
- Определены API endpoints
- Разбивка на 5 фаз реализации
